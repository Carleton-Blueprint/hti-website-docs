In Contentful, each type of page or object (event, blog post, resource, etc.) is defined by a **Content Type**.

## ‚úèÔ∏è Editing a Content Model

1. Go to the **Content Model** tab.
2. Select a content type (e.g., `event`, `blogPost`).
3. Add, remove, or modify fields (e.g., `title`, `image`, `description`).
4. Add validation rules (e.g., required fields, unique slugs).

| Type          | Purpose                         |
|---------------|----------------------------------|
| `event`       | Events listing and detail pages  |

| `resource`    | Tips, tools, and external links  |
| `teamMember`  | Team bios, roles, and photos     |
